System Requirements

Operating System: Windows based (e.g. Windows 7, Windows Server 2016 or higher)
Database: PostgreSQL (11/12/13)
Hardware Requirements: No Minimum; Recommendation: 2 CPU cores and 4GB RAM
HDD/Space: This depends on the data you would like to store and share. Most instances run with 100GB HDD and still have enough free space

FUSION: Heinz-Nixdorf Endowed Chair for Distributed Information Systems
Friedrich Schiller University of Jena
